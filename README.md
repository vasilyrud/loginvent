loginvent
=========

NYU Abu Dhabi IDEA Lab Experimental Inventory System

This prototype was tested with:
* Ubuntu 14.04.2 LTS
* PHP 5.5.9-1ubuntu4.9
* PostgreSQL 9.3.7
* Bootstrap 3.2.0
* RequireJS 2.1.14
* jQuery 1.11.1

Plugins/Libraries used:
* FPDF 1.7
* Google-API-PHP-Client
* PHPQRCode 1.1.4
* jQuery MD5 Plugin 1.2.1
* jsQRCode
* QR Decoder
* JQuery UI 1.11.0
* RequireJS domReady 2.0.1
* list.js
